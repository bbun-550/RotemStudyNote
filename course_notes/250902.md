#### Bagging Bootstrap Aggregating

- bagging
	- 동일 데이터 형태 예측 조합
	- 병렬적인 결합으로 트리 모두 독립적으로 운용
	- 분산감소 목적
	- 모든 트리별 동일 가중치

- boosting
	- 직렬적 결합, 트리별 전후 관계가 중요하다.
	- 트리별 성능 기반 차별적 가중치
