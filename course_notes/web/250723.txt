### git fork
> Fork : 다른 사용자의 저장소를 내 계정으로 복사하는 과정
>- 원본 저장소와 독립적으로 관리
>##### 포크 과정
>- 원본 저장소 fork : 다른 사용자의 저장소에서 fork하면, 내 계정에 복사가 되어 자유롭게 수정하고 개발할 수 있다.
>- 로컬에서 clone : 포크한 저장소를 로컬로 가져오기 위해 git clone
